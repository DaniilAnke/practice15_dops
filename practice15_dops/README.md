Проект: Бинарное дерево поиска (BST)
Это простое решение на Python, которое позволяет реализовать классические операции с бинарным деревом поиска: вставка, поиск, удаление, вычисление высоты и проверку сбалансированности.

------------------------------------------Требования

Для запуска приложения необходимы:

Python версии >= 3.8

------------------------------------------Установка

Нет сторонних зависимостей:

git clone https://github.com/your_username/binary-search-tree.git

------------------------------------------Запуск и тестирование

-----------------------Создать экземпляр класса BinarySearchTree

from src.binary_search_tree import BinarySearchTree

# Создание экземпляра дерева
bst = BinarySearchTree()

-----------------------Добавьте элементы:

bst.insert(50, 'Value of 50')
bst.insert(30, 'Value of 30')
bst.insert(70, 'Value of 70')

-----------------------Проверьте работу функций:

print(bst.search(30))  # Выведет: Value of 30
bst.delete(30)
print(bst.search(30))  # Выведет: None

-----------------------Тестирование:

Запустите тесты с использованием стандартной библиотеки Python:

python -m unittest discover src/tests
